THE FIVE STATUS LEVELS (in order of permanence):
═══════════════════════════════════════════════════════════════════════

1. SHORT VISIT (tourist, business trip, conference)
   Houses: H3 (short journey) + H9 (long journey, if overseas) + H11 (purpose)
   Dasha: any AD signifying H3 + H9 briefly
   Duration: weeks to months
   Indicator: H9 active without H12 strong = visit only, no stay

2. WORK VISA / STUDY VISA (temporary status)
   Houses: H6 (service) or H4 (education) + H9 + H12
   H12 sub lord must be activated for the stay component
   H11 connection = visa approved successfully
   Dasha: AD signifying H6/H4 + H9 + H12 simultaneously
   Duration: 1-6 years typically

3. PERMANENT RESIDENCY (PR / Green Card / equivalent)
   Houses: H4 (permanent home) + H9 (long journey) + H11 (gain) + H12 (foreign land)
   ALL FOUR must be signified for PR to actually be granted
   H4 is the differentiator — without H4, native is on temporary visa only
   Dasha: AD signifying H4 + H11 + H12 for the application timing
   Specific PR timing: when AD AND PD both signify H4 + H11 + H12 = grant date

   Common error: predicting PR when only H9 + H12 are signified (that is just
   work-stay continuation). H4 must enter the chain for PR specifically.

4. CITIZENSHIP (full naturalization)
   Houses: H4 + H9 + H10 + H11 + H12 (most layered status)
   H10 = formal status (citizenship has formal rights)
   H4 = permanent home
   H11 = fulfillment of permanence
   Dasha: AD signifying ALL FIVE, often with Jupiter or Saturn involvement
   Citizenship typically comes 5-10 years after PR; check next major AD shift
   after PR-grant period.

5. RETURN TO HOMELAND (reverse migration)
   Houses: H4 (homeland) becoming activated WHILE H12 weakens
   Specifically: H4 sub lord signifying H4 + H2 + H11 = return home with gain
   Saturn AD often triggers return-home if Saturn signifies H4
   Dasha pattern: any AD where H12 is no longer activated AND H4 is
