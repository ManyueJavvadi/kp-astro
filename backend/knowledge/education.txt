ABROAD EDUCATION / STUDY OVERSEAS (PR A2.6)
================================================================================

ABROAD STUDY PROMISE:
  H4 (school) or H9 (higher learning) + H12 (foreign land) + H11 (gain)
  ALL THREE must connect for abroad-study to materialize
  For PhD/research abroad: H9 primary + H12 + H11
  For undergraduate abroad: H4 primary + H12 + H11

VISA / ADMISSION TIMING:
  Scan ADs for H12 + H9 (or H4) + H11 alignment
  Jupiter favorable in chain = formal admission, structured program
  Mercury favorable = test scores good (GRE/GMAT/SAT/IELTS/TOEFL)
  Rahu favorable = foreign / overseas alignment strong
  Venus favorable = cultural fit / quality-of-life favorable
  Sun favorable = government scholarship / authority-supported

SPECIFIC TEST OUTCOMES (GRE / GMAT / CAT / UPSC / SAT):
  H3 sub lord favorable + H6 (competitive exam) + H11 (success)
  Mercury strong = analytical test sections succeed
  Jupiter strong = qualitative / essay sections succeed
  H4 + H11 → admission follows successful test scores

SCHOLARSHIP / FINANCIAL AID PROBABILITY:
  H11 (fulfillment of desire) + H2 (received money) + Jupiter favorable
  Sun involvement = government/merit-based scholarship
  H4 sub lord signifies H2 AND H11 = scholarship covers tuition
  Without H11+H2 favorable in chain = self-funded / loan-based study

FAILURE-RECOVERY PATH:
  If a chart shows the abroad-study window OPENS but native fails
  test/visa: scan NEXT 2 favorable ADs. KP timing windows often
  repeat structurally; the next firing can be retake-success.
