**Purpose**: Combined KP doctrine for public-eye careers — fame /
celebrity / influencer; politics / election / public office; sports /
athletic competition.

**Source**: KSK Reader V (career + status sections) + KP Astrology

(king-yogas) + modern political-career analysis literature + sports
astrology research.

**Sensitivity tier**: TIER 1 default. TIER 2 for:
- Politics with violence-risk context (controversial campaigns)
- Sports career-decision under injury pressure
- Fame-pursuit driven by mental health vulnerability (cross-link
  to mental_health.md)

**Engine support**: Topics `fame`, `politics`, `sports` (canonical)
+ ~30 aliases route here via PR B5.0a + B5.3.

## 1. THE COMMON FOUNDATIONAL FRAMEWORK

All three domains (fame / politics / sports) share core KP houses
because they all involve PUBLIC ACTION:

### Shared core houses

- **H10** — career / authority / public status / what the public sees
- **H11** — gain / community / followers / electoral support
- **H1** — self-projection / personality / individuality
- **H3** — courage / initiative / communication

### Shared core karakas

- **Sun** — authority / king / leadership / spotlight
- **Jupiter** — wisdom / public respect / dharmic leadership
- **Mars** — energy / drive / competition
- **Mercury** — communication / strategy / media
- **Venus** — charisma / entertainment-appeal (fame especially)
- **Saturn** — sustained discipline / long-arc career

### Common denial houses

- **H6** — defeat / public criticism / scandal-as-disease
- **H8** — sudden downfall / hidden disruption
- **H12** — loss / exile / privacy invasion

## 2. FAME / CELEBRITY / INFLUENCE

### Specific KP signature for fame

H10 + H11 + H1 + H5 favorable with Sun + Jupiter karakas →
structural fame promise.

### Fame TYPE by chart character

**Acting / cinema / theater**:
- H5 (entertainment / creative) + H10 + Venus + Moon
- Venus favorable = visual / cinematic appeal
- Moon favorable = mass-emotional connection

**Music / performing arts**:
- H5 + H10 + Venus + Saturn (discipline-of-craft)
- Mercury favorable = lyrical / composition skill
- Moon favorable = emotional resonance

**Writing / journalism / authorship**:
- H10 + H3 (communication) + Mercury favorable
- Jupiter favorable = literary depth
- Saturn favorable = sustained writing discipline

**Social media / influencer / YouTuber (modern)**:
- H10 + H11 + Mercury (digital communication) + Rahu (unconventional
  fame route) + Venus (appeal)
- Mercury + Rahu favorable = digital virality
- H5 (creative content) + H11 (audience growth)

**TV / news anchoring / public speaking**:
- H10 + H3 + Mercury + Sun (authority of voice)
- Jupiter favorable = credibility

**Sports celebrity** (cross-link to §4 below):
- H10 + H5 + Mars + Sun

### Fame TIMING

Standard fame-window scan:
1. Identify AD where Sun + Jupiter + H10 + H11 align
2. Cross-check with Rahu transit (Rahu often correlates with
   sudden viral fame)
3. Saturn favorable = sustained fame; without Saturn = flash fame

### Sustainable fame vs flash fame

- **Sustainable**: Saturn favorable + Jupiter favorable + H10 sub
  lord signifies H11 cleanly across multiple ADs
- **Flash fame** (one viral moment, no sustained career):
  Rahu involvement + H8 (sudden) + H10 active in a single AD;
  fades quickly when AD shifts
- **Late-career fame** (recognition after years of work): Saturn
  dasha + H10 + H11 (the "overnight success after 20 years"
  pattern)

### Fame's dark side — public criticism / cancellation / scandal

- H10 + H6 affliction = professional criticism / cancellation risk
- H10 + H8 + Rahu = sudden scandal / leaked information
- H10 + H12 = privacy invasion / paparazzi pressure
- Recommend (during such windows): PR consultation + legal counsel +
  mental health support + lifestyle privacy practices

### Fame's mental-health impact

- Sustained public attention strains mental health
- Cross-link to `mental_health.md` for celebrities asking about
  depression / anxiety / addiction
- Common pattern: H10 strong + Moon afflicted = public-success +
  private-struggle

## 3. POLITICS / ELECTION / PUBLIC OFFICE

### Specific KP signature for political career

H10 + H11 + H7 favorable with Sun + Jupiter karakas → structural
political promise.

### Why H7 matters for politics

H7 = the public / opponents in democratic contest. Politicians need
H7 + H11 alignment (public + mass support) to win elections.

### Election outcome doctrine

For specific election prediction questions:

**WIN signature**:
- H10 sub lord signifies H10 + H11 (status + mass support)
- H11 sub lord signifies H1 + H10 + H11 (self + status + gain)
- H6 sub lord signifies H1 + H6 + H11 (overcoming opponents)
  (this is the litigation-win doctrine applied to election-win)
- Sun strong (king-yoga) + Jupiter favorable (mandate)
- AD lord favorable on election date
- Ruling Planets on election day include H10/H11 significators

**LOSS signature**:
- H7 sub lord stronger than H1 (opponent stronger)
- H10 sub lord signifies denial set (H5/H8/H9/H12 with no relevant)
- H6 favoring opponent (Mars on H6 + H7 = opponent's aggressive
  campaign succeeds)

### Election TIMING

For "will I win this election":
1. Identify exact election date
2. Compute MD/AD/PAD/Sookshma for that date
3. Apply WIN signature above
4. Cross-check with national chart (mundane astrology layer —
   beyond pure individual KP)
5. Provide structural probability: HIGH / MODERATE / LOW

### Tier escalation for political questions

Politics is INHERENTLY ADVERSARIAL. Sensitive contexts:

**Violence-risk campaigns**:
- H10 + H6 + Mars + Saturn affliction = controversial / contentious
- Recommend security + media-strategy + family-protection
- TIER 2 framing

**Imprisonment risk** (politicians under legal scrutiny):
- Cross-link to `litigation.txt` §5 (jail risk) — TIER 3
- Politicians facing CBI/ED cases need litigation framework

**Suicide risk after electoral loss**:
- Some politicians (especially first-time candidates with personal
  investment) face suicide risk after losing
- Cross-link to `mental_health.md` §3 — TIER 3 ABSOLUTE if any
  crisis-language

### Political career LONG-ARC

For "will I ever hold public office" vs "should I run this term":
- Long-arc: scan upcoming 10-15 years for Sun + Jupiter ADs with
  H10 + H11 strong
- Specific election: focus on current AD/PAD + transit overlay
- Recommend: political mentor + party affiliation + community
  building (not just chart-timing)

### Activist / political dissent

For activist-type questions (not seeking office, but public-cause work):
- H9 (dharma / cause) + H11 (community) + H10 (visibility)
- Jupiter favorable = dharmic cause
- Saturn favorable = sustained activism
- Mars + H6 + H11 = combative activism (effective but draining)

## 4. SPORTS / ATHLETIC COMPETITION

### Specific KP signature for sports career

H5 + H6 + H11 favorable with Mars + Sun karakas → structural sports
promise.

### Why these houses

- **H5** — sports = entertainment + play + competition + speculation
- **H6** — competition / overcoming opponents / endurance
- **H11** — winning / gain from competition / trophies

### Sports TYPE by chart character

**Cricket / team-bat sports** (India primary):
- H5 + H6 + Mercury (strategy) + Mars (energy)
- Bowler vs batter distinction: Mars dominant = bowler;
  Mercury + Moon = batter (timing-feel)

**Football / soccer / team-field sports**:
- H5 + H6 + Mars + H11 (team-coordination)
- Mars strong + Jupiter favorable = leadership role on team

**Tennis / badminton / individual racket**:
- H5 + H6 + Mercury + Mars + Saturn (precision endurance)
- Mercury dominant = strategy player; Mars = aggressive player

**Chess / mind sports**:
- H5 + Mercury (pure) + Jupiter (depth) + Saturn (endurance)
- Mars secondary

**Boxing / combat / wrestling / martial arts**:
- H5 + H6 + H8 (intense physical confrontation) + Mars (primary)
- Saturn favorable = championship-level endurance

**Athletics / track-field**:
- H5 + H6 + Mars + Sun (sprint / sun-energy) or Saturn (marathon /
  long-arc endurance)

**Olympics / international competition**:
- H5 + H6 + H11 + H9 (international / abroad) + H12 (foreign-setting)
- Sun favorable = top-tier representation
- Cross-link to `foreign.txt` for international travel context

### Game result prediction

For "will I win this match" / "who will win the championship":

This is a KP Prashna context — the question moment matters:
1. Cast Prashna chart at moment of question
2. Read Prashna's H1 (self) vs H7 (opponent)
3. H1 stronger + H11 favorable → native wins
4. H7 stronger + H6 favorable for opponent → opponent wins
5. Mercury favorable + Mars favorable for the side asking → that
   side has structural support

### Sports career TIMING

For "should I pursue sports professionally":
- Identify favorable youth-AD (typically Mars or Mercury or Sun)
- H5 + H6 + H11 + H10 alignment in that AD
- Saturn favorable across LATER dashas = sustained career
- Without Saturn favorable = short career (peak then injury-or-shift)

### Injury risk

- H5 + H6 + Mars + H8 (sudden injury) + adverse transit
- Cross-link to `health.txt` §10 (accident risk — sports-specific
  injury subsection)

### Career transition out of sports

- Sports career typically peaks in 20s-30s (most disciplines)
- Read post-sports career: H10 sub lord activation in later AD
- Common transitions: coaching (H10 + H4 mentor role), commentary
  (H3 + H10), business (H7 + H10 — sports-related business)

## 5. PUBLIC-EYE COMMON ISSUES (all three domains)

### Imposter syndrome / fear of being "found out"

- H10 strong + H1 weak + Moon afflicted = success without inner
  alignment
- Common in early-career public figures
- Recommend: therapy + mentor + spiritual practice
- Cross-link to `mental_health.md`

### Public-private divide

- H10 (public) vs H4 (private home) tension
- Mars or Saturn afflicting H4 in public-career window = home life
  stressed by career
- Recommend: family agreements + boundaries + scheduled family time
- Common pattern: H10 strong / H4 weak = career-driven, family-strained

### Social media + mental health

Modern context — public figures are bombarded by online
criticism:
- Mercury + Rahu (digital communication) + H6 (criticism) + H12
  (hidden trolls)
- Recommend: social media management policies + mental health
  hygiene + therapy if needed
- Cross-link to `mental_health.md` §2.2 (anxiety) and §2.7
  (PTSD if doxxing/threats)

### Financial pressure on public figures

- Salary doesn't always match public-status
- Politicians: financial pressure to fund campaigns
- Athletes: peak earnings concentrated in short career
- Recommend: financial planner + diversified income strategies
- Cross-link to `wealth.txt`

## 9. WORKED EXAMPLES (quick reference)

### "I want to be an actor — does my chart support fame?"

**Approach**:
1. Tier 1 NOTE
2. Read H5 + H10 + Venus + Moon
3. Identify actor-specific signature (§2)
4. Recommend domain-fit (cinema / theater / web series)
5. Identify favorable career-start AD
6. Sustainability check (Saturn favorable across multiple ADs)
7. Recommend: acting school + auditions + community + mental health
   awareness about industry pressures

### "Will I win the upcoming MLA election?"

**Approach**:
1. Tier 1 NOTE (Tier 2 if violence-context)
2. Read H10 + H11 + H7 for election date
3. Apply WIN signature (§3)
4. Cross-check with opponent's chart if known
5. Mundane astrology layer (national / state context — caveat that
   KP individual chart is one factor among many)
6. Specific probability (HIGH / MODERATE / LOW)
7. Recommend: campaign strategy + party support + security

### "Will my son make it to the cricket national team?"

**Approach** (Bhavat Bhavam — reading via father's chart):
1. Tier 1 NOTE
2. Confidence floor ~70% via Bhavat Bhavam
3. Read native's H5 (= son's H1) + H10-via-Bhavat Bhavam
4. Sports signature applied via Bhavat Bhavam (§4)
5. Recommend: son's own birth data for ~95% precision
6. Identify favorable cricket-career window
7. Recommend: academy + coaching + sustained practice +
   parent-as-supporter role

### "Should I post controversial political opinions online?"

**Approach**:
1. Tier 2 NOTE (potential consequences)
2. Read H10 + H3 (communication) + H6 (criticism risk)
3. Mercury + Rahu (digital) + H12 (trolls)
4. Assess: is current period H10-supportive enough to handle backlash?
5. Decision-support framing (cross-link to §5 below + decision_support.md)
6. Recommend: think-through career impact + family consideration +
   media training before posting

### "I lost the election badly. What's next?"

**Approach**:
1. Tier 2-3 NOTE (potential mental health impact)
2. Acknowledge the loss
3. Read recovery dasha (next favorable H10 / H11)
4. Re-election prospect if interested
5. Career-pivot options (consulting / activism / writing)
6. CRITICAL: cross-link to `mental_health.md` for grief / depression
   support
7. Crisis resources if any suicide-language
